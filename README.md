<h1>Car Commerce</h1>
Next.js, with Tailwind CSS and some ESLint rules

## Getting Started

1. Clone the repo
```
git clone https://github.com/Albertadi/carcommerce.git
```
2. Install python dependencies
```
pip install -r requirements.txt
```
3. Install NPM Packages
```
npm install
```
4. Start backend
```
python app.py
```
5. Start frontend
```
npm run dev
```
